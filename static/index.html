Welcome to Amiibo RESTful API by N3evin & cheatfreak47

List of amiibo

/api/v1/amiibo

  • return: JSON format of all the amiibo in the database.
Get specific amiibo detail

/api/v1/amiibo/{value}

  • value: can be the name of the amiibo, head value or tail value. Head or Tail value must be 8 digit.
  • return: JSON format of a list of amiibo belong to the value.
Get the list of amiibo base on the type.

/api/v1/amiibo/type/{type}

  • type: The type, figure, card or yarn. Can be their hexdecimal value. Hex example: 0x2
  • return: JSON format of all the amiibo belong to the type.
Get the list of amiibo base on game series.

/api/v1/amiibo/gameseries/{value}

  • value: The game series. Example: The Legend Of Zelda
  • return: JSON format of all the amiibo belong to the game series.
Get the list of amiibo base on amiibo series.

/api/v1/amiibo/amiiboseries/{value}

  • value: The amiibo series. Example: Legend Of Zelda
  • return: JSON format of all the amiibo belong to the amiibo series.
Get the list of amiibo base on character.

/api/v1/amiibo/character/{character}

  • character: The amiibo character. Example: Zelda
  • return: JSON format of all the amiibo belong to the amiibo series.
Get all the characters in the database

/api/v1/character

  • return: JSON format of a list of characters available in the database.
Get the details of certain character

/api/v1/character/{value}

  • value: Can be both name or key value.
  • return: json format of all the character belongs to the value.
Get all the types of amiibo in the database

/api/v1/type

  • return: json list of all the types in the database.
Get a type information

/api/v1/type/{value}

  • value: Can be both name or key value that you wanted.
  • return: json list of the types in the database.
Get all the amiibo series in database.

/api/v1/amiiboseries

  • return: json list of all the amiibo series in the database.
Get an amiibo series in database.

/api/v1/amiiboseries/{value}

  • value: Can be both name or key value that you wanted.
  • return: json list of the amiibo series in the database.
Get all the game series name and code in database.

/api/v1/gameseries

  • return: json list of the game series in the database.
Get the information for game series. [Type Name]

/api/v1/gameseries/{name}

  • name: The name of the series.
  • return: json list of the key belong to that series.
Get the information for game series. [Type Key]

/api/v1/gameseries/{key}

  • key: The key of the series.
  • return: JSON of the name belong to that key.
