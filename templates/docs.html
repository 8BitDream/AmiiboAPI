    Amiibo

    It returns information of all amiibos or specific amiibo.

    GET /amiibo/

    Return a list of amiibo available in the API.

    GET /amiibo/{id or name}

    Note: id could be either head or tail or head and tail.

    Amiibo

    Name Description Data Type
    amiiboSeries The series the amiibo belongs to. string
    character The character of the amiibo, multiple character have different amiibo design. string
    gameSeries The game series of the amiibo. string
    head The 1st 8 value of the hexadecimal to recognize the amiibo.
    Note: 0-7 value in the hexadecimal    		 string
    image The image link of the amiibo. string
    name The name of the amiibo. string
    tail The next 8 value of the hexadecimal to recognize the amiibo.
    Note: 8-15 value in the hexadecimal    		 string
    type The type it belongs to. Example: card, figure and yarn string

    Amiibo Type

    Get all the amiibo based on it's type.

    GET /amiibo/type/{type or type id}

    Note: type id could be anything in the type database. Eg. 0x2 (Yarn)

    {
	"amiibo": [
	{
		"amiiboSeries": "Yoshi's Woolly World",
		"character": "Yoshi",
		"gameSeries": "Super Mario",
		"head": "00030102",
		"image": "https://raw.githubusercontent.com/Falco20019/libamiibo/master/libamiibo.images/Images/icon_0003010200410302.png",
		"name": "Green Yarn Yoshi",
		"tail": "00410302",
		"type": "Yarn"
	},
	...
]
}

    Amiibo Game Series

    Get all the amiibo based on it's game series.

    GET /api/v1/amiibo/gameseries/{gameseries or gameseries id}

    Note: gameseries id could be anything in the type database. Eg. 0x2 (Chibi Robo)
